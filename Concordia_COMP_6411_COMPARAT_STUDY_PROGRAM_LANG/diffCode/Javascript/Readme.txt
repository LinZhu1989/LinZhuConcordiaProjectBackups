Potum-The Beverage Coaster Project can be implemented by double-clicking POTUM_HTML.html file which is an HTML file which has the reference to the JavaScript file which is POTUM.js which has the actual script for contains the POTUM algorithm to calculate the vale of "L".

By doing the above, the implementation is displayed in a standard compatible browser such as Google Chrome,Mozilla Firefox,Internet Explorer etc.,

A Text Box with a Label "Enter the value of R" is displyed, click the button "Calculate L" to successfully calculate the Length of overlap of two circles (L). Also displaying the Pie value, Alpha value which are used for computations in calculating the "L" as per POTUM algorithm.

Note:
The POTUM_HTML.html and POTUM.js to be in same location/folder.
