Below are the files used for POTUM implementation in python language:

constants.py : Module calculating value of math functions used in POTUM

potum.py     : Module calculating value of L for which area of overlapping coaster is half the area of any two of the coaster


Getting Python:
----------------
The most up-to-date and current source code, binaries, documentation, news, etc. is available at the official website of Python:

Python Official Website : http://www.python.org/


Install Python on Windows:
---------------------------
Here are the steps to install Python on Windows machine.

Open a Web browser and go to http://www.python.org/download/

Follow the link for the Windows installer python-3.3.2.msi file.

Run the downloaded file by double-clicking it in Windows Explorer.


Use IDLE ("Interactive DeveLopment Environment") for running .py file:
---------------------------------------------------------------------
Click on Windows start button and browse to Python > IDLE (Python GUI)

File > Open (Ctrl+O) to open potum.py file

Press F5 to run potum program
