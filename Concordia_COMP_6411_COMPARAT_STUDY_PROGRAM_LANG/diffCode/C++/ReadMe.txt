////////////////////////***** Compliling Instructions for C++****//////////////////////////////

Tools needed: 

Microsoft Visual Studio, Code::blocks, Microsoft Visual C++, Eclipse

Compiling Instructions on Visual studio:

1) Create a new project.
2) Click on add files in source files. 
3) Copy the code paste it into a .cpp file.
4) Press F5 to debug. 
5) You can now see the output.
